Local Mode Setup Instructions

Follow these simple steps to run Vega AI on your computer

Quick Setup Guide

1

Create a config file

First, get your free Gemini API key. Then create a file named config with:

GEMINI_API_KEY=your-api-key-here

Replace your-api-key-here with your actual key.

2

Run Vega

Run this command in the same folder as your config file:

docker run -d --name vega-ai -p 8765:8765 --env-file config ghcr.io/benidevo/vega-ai:latest

This starts Vega with your API key configured.

3

Start Using Vega

Visit http://localhost:8765 and log in with:

Username: admin

Password: VegaAdmin

Security Notice: Change your password immediately after first login

Go to http://localhost:8765/settings/profile to update your credentials
